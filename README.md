# Opis projektu

Projekt umożliwia szybkie uruchomienie animacji wykresu funkcji i jej pochodnej zależnej od trzech parametrów. Do jego działania wymagane jest zainstalowanie MATLAB-a wraz z odpowiednimi pakietami.

# Działanie aplikacji

## Konfiguracja

- Po uruchomieniu programu należy wprowadzić funkcję, której argumentem jest `x`.
- Możliwe jest uwzględnienie do trzech parametrów: `a`, `b` oraz `c`.
- W polu **Przykłady** można wybrać jedną z przykładowych funkcji istotnych z punktu widzenia fizyki.

## Dostosowywanie osi wykresu

- W polach **Zakres osi X** oraz **Zakres osi Y** możliwe jest ręczne ustawienie zakresów osi.
- Pole w lewej dolnej części interfejsu pozwala dostosować ustawienia zakresów osi. Opcja **Skala osi Y dla obu wykresów** umożliwia automatyczne dostosowywanie zakresów osi do wyświetlanych danych.

## Zmiana parametrów i animacja

- Suwaki w polu **Parametry** pozwalają zmieniać wartości parametrów w ustawionych zakresach.
- Przycisk **Równoczesne ustawianie** umożliwia zaznaczenie parametrów, które mają zmieniać się jednocześnie po przesunięciu jednego z suwaków.
- Wciśnięcie przycisku **Animacja** uruchamia animację, w której wybrane parametry zmieniają się w zadanych zakresach.
