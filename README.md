# Opis projektu
Projekt umożliwia szybkie uruchomienie animacji wykresu funkcji i jej pochodnej zależnej od tzrech parametrów. Do jego działania wymagane jest zainstalowanie Matlaba wraz z odpowiednimi pakietami.

# Działanie aplikacji
## Konfiguracja
- Po uruchomieniu programu należy wprowadzić funkcję, której argumentem jest x.
- Możliwe jest uwzględnienie do trzech parametrów: a, b lub c
- W polu przykłady można wybrać jedną z przykłądowych funkcji ważnych z punktu widzenia fizyki
## Dostosowywanie osi wykresu
- W polach "zakres osi x" oraz "zakeres osi y" możliwe jest ręczne ustawinie zakresów osi
- Pole w lewej dolnej części interfejsu pozwala dostosowac ustawienia zakresów osi. Opcja "Skala osi y dla obu wykresów" umoliwia ustawienie dostosowywania zakresów osi do danych
## Zmiana parametrów i animacja
- Suwaki w polu "Parametry" pozwalają zmieniać wartości parametrów w zakresach, które można ustawić
- Przyciska "Równoczesne ustawianie" umożliwia zaznaczenie parametrów, które mają się zmieniać równocześnie po przesunięciu jednego z nich
- Wciśnięcie przycisku "Animacja" uruchamia animację, w której wybrane parametry zmieniają się w ustawionych zakresach
